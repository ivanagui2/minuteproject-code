To see the entire generation suite in all the technologies delivered:

1) Start the sample database:
In <MP_HOME>/sample
Open a command prompt
Type start-petshop-database.cmd for Windows
Type ./start-petshop-database.sh for Linux

2) Execute the generations
In <MP_HOME>/demo/config 
Open a command prompt
Type all-demos.cmd for Windows
Type ./all-demos.sh for Linux
