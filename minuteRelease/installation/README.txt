To start with minuteproject:

1) try the demo application by following the instructions in /demo/config/README.txt
It will show you how to create backend based on Spring/Hibernate/ibatis/JPA
It will also provide you methodology fixtures if you have to good habit to work in BDD with FitNesse.

2) If you want to customize your generated code, you can find a short introduction in /mywork/README.txt
